Why this is a different layer

Eval is detection. This is prevention.

Tracing and evaluation tell you the agent misfired after it happened. A contract is a deterministic pre/post predicate that stops the action at the boundary.

Deterministic where it can be.

Most failure modes are expressible as code — no second LLM call, no latency tax, no flakiness. The ones that aren't fall back to a judge model.

Trained on real corrections.

The failure taxonomy isn't hypothetical. Each contract maps to an actual repeated failure observed in a live agent and the recovery that fixed it.

Signed receipts.

Every gated action emits a tamper-evident receipt — an audit trail your compliance team can actually read.

Wrap any agent loop in ten lines.

pip install -e git+https://github.com/impartshadow/shadow-kit.git#egg=shadow-kit

from shadow_kit.contracts import ContractContext, check_all_pre, check_all_post

ctx = ContractContext(action="respond", response_text=agent_output)
violations = check_all_post(ctx)
if any(v.severity == "block" for v in violations):
    agent_output = retry_with_feedback(violations)   # correct before it ships
